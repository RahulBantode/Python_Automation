---Directory/File traversal---
---Usage and command line input to the program---

  File path should be absoulute 
  without arguments - <executable>
  Traverse  - <executable> <absoulute path name>
  File size - <executable> <absoulute path name> <-s>
  Find File - <executable> <absoulute path name> <-f> <file substring>
  File size greater than - <executable> <absoulute path name> <-s> <size_of_file in byte>
  File size and extensin - <exe> <abs path name> <-s> <-f> <size> <exten/substring>

  NOTE / All the file arguments should be in above format because this is command line input
